# AI-Powered Human Identification System

Final Year Project — BS Computer Science

## What This System Does
A web platform where families report missing persons,
hospitals report unidentified patients, and police 
manage wanted individuals. AI face matching connects
these records using deep learning.

## Tech Stack
- Frontend: Next.js + Tailwind CSS
- Backend: Python Flask  
- AI: DeepFace + FaceNet
- Database: SQLite
